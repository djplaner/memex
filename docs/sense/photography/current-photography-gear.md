## Configuration

### Olympus OM-D E-M5 Mark II

| [[photoclass-2026]] mode | OM-D E-M5 Mark II mode | Notes |
|--------------------------|-------------|-------|
| Auto mode | Point-and-shoot (**P**) Mode | Set it using the mode dial to **P**. Shutter speed and aperture will blink if camera unable to achieve correct exposure |
| Aperture priority mode | Aperture Priority (**A**) Mode | Set it using the mode dial to **A** or **Av**. You control the aperture, the camera adjusts shutter speed. |
| Shutter priority mode | Shutter Priority (**S**) Mode | Set it using the mode dial to **S** or **Tv**. You control the shutter speed, the camera adjusts aperture. |
| Manual mode | Manual (**M**) mode | Set it using the mode dial to **M**. You control both aperture and shutter speed. Will show the difference from correct exposure. _BULB_ and _TIME_ shutter speed values keep the shutter open based on interactions with the shutter button |
