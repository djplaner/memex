[Vite plugin server](https://github.com/lisonge/vite-plugin-monkey) to build userscripts, including support for svelte


## Usage

1. `npm i -D vite-plugin-monkey`
1. `npm create monkey`
2. Choose options, including name
3. `npm install`
4. `npm run dev`
